# Kut

Kut language D interpreter

IMPORTANT: Main development (on C) is [here](https://github.com/erenjanje/kut)

# Examples

### Hello World

```(ekran "Merhaba, Dünya" yazsın)```

### Variable Creation

```('değişken 5 olsun)```

### Basic Arithmetic

```(((((5 2 +) 3 *) 10 -) 2 /) 6 ^)```

### Combined

```
('x 10 olsun)
('y 8 olsun)
(ekran (x y +) yazsın)
```
